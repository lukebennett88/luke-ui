`vars` is the typed public semantic-token contract. Each typed path maps to one stable global CSS
custom property by kebab-casing its segments under the `--luke-` prefix. For example,
`vars.color.intent.danger.surface.solidHover` resolves to
`var(--luke-color-intent-danger-surface-solid-hover)`.
