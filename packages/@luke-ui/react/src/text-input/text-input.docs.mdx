`TextInput` is a primitive input control built from React Aria `Group` + `Input`.

## Import

```ts
import { TextInput } from '@luke-ui/react/text-input';
```

## Usage

Use `TextInput` inside React Aria `TextField` composition.

```tsx
import { Field } from '@luke-ui/react/field';
import { TextInput } from '@luke-ui/react/text-input';
import { TextField } from 'react-aria-components';

<TextField name="email">
	<Field label="Email" description="We'll only use this for updates.">
		<TextInput placeholder="name@example.com" />
	</Field>
</TextField>;
```

## Adornments

```tsx
<TextInput adornmentStart={<SearchIcon />} adornmentEnd="⌘K" />
```

Adornments accept any `ReactNode`. Consumers are responsible for semantics if
adornments are interactive.

## Size Behavior

`TextInput` uses design-system `size` values.

| Prop | Type                  | Default    | Notes                                   |
| ---- | --------------------- | ---------- | --------------------------------------- |
| size | `'small' \| 'medium'` | `'medium'` | Controls control height and typography. |

The HTML numeric `<input size>` attribute is intentionally omitted. On this API,
`size` is a design-system visual variant.
