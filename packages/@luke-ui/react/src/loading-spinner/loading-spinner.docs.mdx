`LoadingSpinner` renders progress feedback. Omit `value` for indeterminate (unknown progress); pass `value` for determinate.

## Import

```ts
import { LoadingSpinner } from '@luke-ui/react/loading-spinner';
```

`LoadingSpinner` expects the Luke UI theme class to be applied at app/root level.
See [Getting Started](/docs/getting-started).

## Usage

```tsx
<LoadingSpinner aria-label="Loading" />
```

```tsx
<LoadingSpinner aria-label="Loading profile" value={66} />
```

```tsx
<LoadingSpinner aria-label="Syncing" color="informative" size="small" />
```

## Props

`LoadingSpinner` extends `ComponentProps<'div'>` and adds:

| Prop       | Type                   | Default    | Notes                                                                                              |
| ---------- | ---------------------- | ---------- | -------------------------------------------------------------------------------------------------- |
| `value`    | `number`               | -          | When provided, shows determinate progress in range `[minValue, maxValue]`. Omit for indeterminate. |
| `minValue` | `number`               | `0`        | Lower bound for determinate progress.                                                              |
| `maxValue` | `number`               | `100`      | Upper bound for determinate progress.                                                              |
| `size`     | `'small' \| 'medium'`  | `'medium'` | Spinner size token.                                                                                |
| `color`    | `ForegroundColorToken` | -          | Uses `tokens.foregroundColor`.                                                                     |
