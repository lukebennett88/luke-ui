# Button (primitive)

> Primitive button. See `ButtonProps`.

## Import

```ts
import { Button } from '@luke-ui/react/button/primitive';
```

## Props

Extends [`react-aria-components` `ButtonProps`](https://react-spectrum.adobe.com/react-aria/Button.html).

| Prop      | Type                                              | Default    | Description                                                        |
| --------- | ------------------------------------------------- | ---------- | ------------------------------------------------------------------ |
| `isBlock` | `boolean`                                         | `false`    | Whether the button takes up the full inline size of its container. |
| `size`    | `"medium" \| "small"`                             | `'medium'` | Sets the button size.                                              |
| `tone`    | `"primary" \| "neutral" \| "critical" \| "ghost"` | —          | Visual tone. Controls colour scheme.                               |
